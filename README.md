**SCOTUS petitioner win prediction with XGBoost.**

**Data** from SCDB 2025 release 01: https://scdb.la.psu.edu/data/2025-release-01/

Put these CSVs next to the script: SCDB_2025_01_caseCentered_Citation.csv and SCDB_2025_01_justiceCentered_Citation.csv. They are read as latin-1.

Needs pandas, numpy, xgboost, scikit-learn, matplotlib, seaborn. Optuna and shap are optional.
